Kubernetes Static Website Demo

A simple static website built with HTML, CSS, and JavaScript, containerized using Docker, and deployed on Kubernetes (Minikube).
This project is designed for learning and practicing Kubernetes fundamentals such as Deployments, Services, and containerized workloads.

Project Overview
This repository demonstrates:

Creating a basic static website
Containerizing the app using Docker and Nginx
Deploying the container to Kubernetes
Exposing the application using a Kubernetes Service (NodePort)
Running everything locally using Minikube

Tech Stack:
Frontend: HTML, CSS, JavaScript
Containerization: Docker (Nginx)
Orchestration: Kubernetes
Local Cluster: Minikube
